Why Use This Tool?

Government portals for passport and visa applications have strict file size and format requirements. A US passport photo upload needs to be under 240KB in JPEG format. Schengen visa portals may require photos under 500KB. Indian passport applications need specific pixel dimensions.

Most free online converters upload your files to their servers — meaning your passport photo, ID scan, or personal documents pass through unknown infrastructure. This tool processes everything inside your browser using the Canvas API. Nothing leaves your device.

How to Reduce Photo Size to 200KB

Many government portals require photos under a specific file size — 200KB, 240KB, or 500KB. Here's how:

  1. Upload your photo using the drop zone above
  2. Select the Compress mode
  3. Enter your target size (e.g. 200 KB)
  4. Click "Compress" — the tool uses binary search to find the best quality at your target size
  5. Download the result

This works for any target: reduce photo to 100KB, 200KB, 240KB, 500KB, or any custom size.

Convert JPG to PNG, PNG to JPG, WebP to JPG

Some government portals only accept JPEG files. Others require PNG. This tool converts between all common image formats:

  • JPG to PNG — when you need transparency or lossless quality
  • PNG to JPG — when the portal requires JPEG format
  • WebP to JPG — when your phone saves photos in WebP but the portal needs JPEG
  • BMP to JPG — when you have a scanned document in BMP format
  • Any format to WebP — for smaller file sizes with good quality

PDF to Image & Image to PDF

Need to extract pages from a PDF as images? Or combine multiple photos into a single PDF for a visa application? Both are supported:

  • PDF → Image — renders each PDF page as a high-quality PNG or JPEG using pdf.js. Choose 1x, 2x, or 3x scale for different DPI levels.
  • Image → PDF — combines one or more images into a single PDF document. Each image becomes a full page.

All processing happens in your browser. PDFs are rendered client-side — no server involved.

Frequently Asked Questions

Is this tool really free?

Yes, completely free with no limits. No sign-up, no watermarks, no hidden fees.

Are my files uploaded to a server?

No. All processing happens inside your browser using the HTML5 Canvas API and pdf.js. Your photos and documents never leave your device.

How do I reduce my photo to exactly 200KB?

Select Compress mode, enter "200" in the target size field, select "KB", and click Compress. The tool uses binary search compression to find the highest quality that fits under 200KB.

Can I convert PDF to images?

Yes. Select "PDF → Image" mode, upload your PDF, choose output format and scale, then convert. Each page becomes a separate downloadable image.

Does this work on mobile?

Yes. Fully responsive — works on iPhone, Android, iPad, and any mobile browser. No app download needed.

What formats are supported?

JPG/JPEG, PNG, WebP, BMP, GIF for images. PDF for document conversion. You can convert between any of these formats.
